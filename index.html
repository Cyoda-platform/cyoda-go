cyoda-go Helm chart repository

Helm repo for Cyoda-platform/cyoda-go — a lightweight, multi-node Go digital twin of the Cyoda platform.

Install

helm repo add cyoda https://cyoda-platform.github.io/cyoda-go
helm repo update
helm search repo cyoda --versions
helm install cyoda cyoda/cyoda --version 0.6.3

Index

The Helm repo manifest is served at index.yaml.
